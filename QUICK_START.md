# 🚀 Portfolio Quick Start Guide

## 📋 Prerequisites
- Node.js 18+ installed
- Git repository set up
- GitHub account

## ⚡ Quick Deployment Steps

### 1. Install Dependencies
```bash
npm install
```

### 2. Build for Production
```bash
npm run build
```

### 3. Deploy to GitHub Pages
```bash
git add .
git commit -m "Deploy portfolio update"
git push origin main
```

### 4. Check Deployment
- Go to your GitHub repository
- Click "Actions" tab
- Wait for "Deploy to GitHub Pages" to complete (green ✅)
- Visit: https://yad4o.github.io/portfolio-web/

## 🎯 What You'll See
- Stunning 4D particle animations
- Morphing geometric shapes
- Scroll-responsive effects
- Professional dark theme

## 🔧 If Something Goes Wrong
1. Check GitHub Actions for errors
2. Verify all dependencies are installed
3. Ensure GitHub Pages is enabled in repository settings
4. Check that base path is set to `/portfolio-web/` in vite.config.ts
